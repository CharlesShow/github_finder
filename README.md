<h1>Projeto GitHub Finder</h1>
<h2>Descrição</h2>
<p>Projeto que busca todos os usuários do GitHub e mostra os 5 melhores repositórios avaliados. A busca é realizada através da própria API do GitHub.</p>
<p>Projeto rodando: <a href="https://charlesshow.github.io/github_finder">Clique aqui</a>.</p>
